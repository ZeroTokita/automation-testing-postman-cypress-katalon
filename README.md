Proyecto universitario de automatización de pruebas usando Postman, Cypress y Katalon Recorder.

# Proyecto de Automatización de Pruebas
Este repositorio contiene el proyecto universitario de automatización de pruebas
realizado sobre la aplicación Automation Exercise.

## Herramientas Utilizadas
- Postman (Pruebas de API)
- Cypress (Pruebas End-to-End)
- Katalon Recorder (Pruebas UI)

## Aplicación Bajo Prueba
https://automationexercise.com

## Estructura del Proyecto
- postman/: Colecciones y entornos de Postman
- cypress/: Scripts y configuración de Cypress
- katalon/: Casos de prueba grabados en Katalon Recorder

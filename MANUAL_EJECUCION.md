# Manual de Ejecución de Pruebas Automatizadas

## 1. Requisitos Previos
- Sistema operativo Windows
- Navegador Google Chrome
- Node.js (versión LTS)
- Postman Desktop
- Extensión Katalon Recorder

---

## 2. Ejecución de Pruebas con Postman

1. Abrir Postman
2. Importar la colección desde /postman
3. Importar el environment
4. Seleccionar el environment
5. Ejecutar cada request o la colección completa
6. Verificar resultados en la pestaña Tests

---

## 3. Ejecución de Pruebas con Cypress

1. Abrir terminal en la carpeta /cypress
2. Ejecutar:
   npm install
3. Abrir Cypress:
   npx cypress open
4. Seleccionar E2E Testing
5. Ejecutar los archivos .cy.js

---

## 4. Ejecución de Pruebas con Katalon Recorder

1. Abrir Katalon Recorder desde Chrome
2. Abrir el proyecto AutomationExercise_Katalon
3. Ejecutar los test cases KC01, KC02 y KC03
4. Verificar resultados y reportes

---

## 5. Evidencias
Las evidencias de ejecución deben ser capturas de pantalla
de cada herramienta durante la ejecución de las pruebas.
